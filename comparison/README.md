
### For Elixir

```elixir

```

### For Rust

You have to install `rust-script`
with `cargo install rust-script`

```ocaml
#!/usr/bin/env rust-script

// Add Rust code here
```

## How to run

```sh
chmod +x ./file_name.{hs|ml|exs|rs}
./file_name.{hs|ml|exs|rs}
```
